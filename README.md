# Rogue Demo

Простая браузерная roguelike игра, написанная на JavaScript с использованием jQuery.

## Описание

Rogue Demo - это простая 2D roguelike игра, где игрок управляет героем, исследует подземелье, сражается с врагами и собирает предметы. Игра создана как демонстрационный проект с процедурной генерацией уровней.

## Особенности

- Процедурная генерация уровней
- Система боя
- Предметы (мечи и зелья)
- Система здоровья героя и врагов
- Звуковые эффекты
- Спрайтовая графика

## Управление

- **W** - движение вверх
- **S** - движение вниз
- **A** - движение влево
- **D** - движение вправо
- **Пробел** - атака

## Игровые элементы

- 🧙‍♂️ **Герой** - главный персонаж с показателем здоровья 100 HP и базовой атакой 10
- 👾 **Враги** - имеют 30 HP и атаку 5
- 🗡️ **Мечи** - увеличивают силу атаки героя
- 🧪 **Зелья** - восстанавливают здоровье героя

## Установка и запуск

1. Склонируйте репозиторий или скачайте файлы
2. Убедитесь, что все файлы находятся в правильных директориях:
   - `index.html` - в корневой папке
   - `js/script.js` - основной скрипт
   - `js/jquery.min.js` - библиотека jQuery
   - `img/` - папка со спрайтами
   - `sounds/` - папка со звуками
3. Откройте `index.html` в веб-браузере

## Технические требования

- Современный веб-браузер с поддержкой HTML5
- Включенный JavaScript
- Поддержка воспроизведения аудио в формате MP3

## Особенности реализации

- Игровое поле: 40x24 клетки
- Размер клетки: 16x16 пикселей
- Общий размер поля: 640x384 пикселей
- Случайная генерация 5-10 комнат
- 3-5 горизонтальных и вертикальных коридоров
- Автоматическое соединение комнат для обеспечения доступности

## Разработка

Проект использует:
- HTML5 для структуры
- CSS3 для стилей
- JavaScript для логики
- jQuery для манипуляций с DOM
- HTML5 Audio API для звуковых эффектов
